# Assets

Static media for VECTOR: logos, diagrams, screenshots and renders used across
the documentation and the README.

## Suggested layout

```
assets/
├── logo/          # VECTOR logo (SVG + PNG exports)
├── diagrams/      # architecture / wiring diagrams
├── screenshots/   # UI, API docs, dashboards
└── renders/       # CAD renders, photos of the robot
```

Please keep large binaries reasonable in size and prefer SVG for diagrams and
logos so they stay crisp and diff-friendly.
