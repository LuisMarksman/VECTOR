# Acknowledgements

VECTOR stands on the shoulders of the open-source community. It contributes
original software, integrations and system architecture, while building on the
following projects. **Each third-party project remains under its own license.**

## AI & language

- [FastAPI](https://fastapi.tiangolo.com/) — high-performance Python web framework
- [Pydantic](https://docs.pydantic.dev/) — data validation and settings
- [Uvicorn](https://www.uvicorn.org/) — ASGI server
- LLM providers such as [Anthropic Claude](https://www.anthropic.com/) and
  [OpenAI](https://openai.com/) (used via optional API integrations)

## Voice

- [openWakeWord](https://github.com/dscripka/openWakeWord) / [Porcupine](https://picovoice.ai/platform/porcupine/) — wake-word detection
- [faster-whisper](https://github.com/SYSTRAN/faster-whisper) — speech-to-text
- [Piper](https://github.com/rhasspy/piper) — local text-to-speech

## Vision

- [OpenCV](https://opencv.org/) — computer vision toolkit
- [Ultralytics YOLO](https://github.com/ultralytics/ultralytics) — object detection
- [MediaPipe](https://developers.google.com/mediapipe) — gesture / pose
- [Tesseract](https://github.com/tesseract-ocr/tesseract) — OCR

## Robotics

- [ROS 2](https://docs.ros.org/) — robotics middleware
- [Nav2](https://navigation.ros.org/) — autonomous navigation
- [MoveIt 2](https://moveit.ros.org/) — motion planning for manipulation

## Messaging & smart home

- [Eclipse Mosquitto](https://mosquitto.org/) — MQTT broker
- [Eclipse Paho](https://www.eclipse.org/paho/) — MQTT clients
- [Home Assistant](https://www.home-assistant.io/) — smart-home platform (optional bridge)

## Embedded

- [ESP-IDF](https://github.com/espressif/esp-idf) — Espressif IoT dev framework
- [Arduino](https://www.arduino.cc/) — embedded framework
- [PlatformIO](https://platformio.org/) — embedded build system
